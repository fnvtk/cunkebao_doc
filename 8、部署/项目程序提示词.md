# 项目程序提示词

## 输出规范

### 1. 项目基础信息
- **项目名称**：明确项目中文名与英文代码库名。
- **运行环境**：Node.js 版本、Java 版本、数据库版本。

### 2. 部署关键点
- **环境检查**：
  - 检查端口占用。
  - 检查 `.env` 文件配置（数据库连接、API 密钥）。
- **启动命令**：
  - 开发环境：`npm run dev` / `mvn spring-boot:run`
  - 生产环境：`pm2 start ecosystem.config.js` / `java -jar app.jar`

### 3. 维护指南
- **日志查看**：`pm2 logs` 或 `tail -f logs/app.log`。
- **更新流程**：`git pull` -> 依赖更新 -> 重启服务。
