## 二、运营岗位详解

### 2.1 用户运营

#### 岗位信息

| 维度 | 内容 |
|:---|:---|
| **岗位定位** | 负责私域用户承接、培育、转化、复购全流程 |
| **性格匹配** | MBTI: ISFJ/ESFJ/ENFJ \| PDP: 考拉型(≥20分) \| DISC: S型(≥18分) |
| **汇报对象** | 项目主理人 |
| **薪资结构** | 底薪8K + 成交提成(10%) |

#### 核心职责与系统功能

| 职责 | 对应系统功能 | 操作频率 | KPI指标 |
|:---|:---|:---|:---|
| 客户需求挖掘 | 微信管理 → 好友详情 | 每日 | 咨询转化率 |
| 重点客户追踪 | 流量池 → 标签筛选 | 每日 | 成交人数 |
| 消息触达 | 工作台 → 消息群发 | 每日 | 触达人数 |
| 社群运营 | 工作台 → 自动建群 | 每周 | 建群数量 |
| 流量分配 | 工作台 → 流量分发 | 每日 | 分配效率 |

#### 每日工作流程

| 时间 | 工作内容 | 系统操作 |
|:---|:---|:---|
| 09:00-09:30 | 查看昨日数据，整理今日重点客户 | 数据中心 → 数据看板 |
| 09:30-11:30 | 百客追踪（重点跟进意向客户） | 微信管理 → 好友列表 |
| 11:30-12:00 | 群消息推送 | 工作台 → 消息群发 |
| 14:00-16:00 | 私信互动+朋友圈点赞评论 | 触客宝 → 聊天界面 |
| 16:00-17:00 | 重客追访（复购引导） | 流量池 → 高价值客户 |
| 17:00-18:00 | 数据复盘，填写日报 | 数据中心 + 飞书日报 |

#### 薪资跃迁体系

| 等级 | 底薪 | 提成 | 能力要求 |
|:---|:---|:---|:---|
| 初级 | 5,000 | 5% | 能独立接待，转化率≥3% |
| 中级 | 8,000 | 8% | 转化率≥5%，复购率≥15% |
| 高级 | 12,000 | 10% | 转化率≥8%，能带新人 |
| 专家 | 15,000 | 12% | 能带团队，人效达标 |

## 三、系统功能与运营场景

### 3.1 获客阶段

| 运营场景 | 系统功能 | 操作步骤 | 预期效果 |
|:---|:---|:---|:---|
| 海报获客 | 流量池 → 场景获客 | 1. 创建海报 2. 生成二维码 3. 投放推广 | 扫码添加好友 |
| 渠道获客 | 分销管理 → 渠道管理 | 1. 创建渠道 2. 分配渠道码 3. 追踪数据 | 渠道带来新客 |
| 社群获客 | 工作台 → 自动建群 | 1. 创建建群任务 2. 选择目标好友 3. 执行建群 | 批量建群 |

### 3.2 养客阶段

| 运营场景 | 系统功能 | 操作步骤 | 预期效果 |
|:---|:---|:---|:---|
| 朋友圈运营 | 工作台 → 朋友圈同步 | 1. 选择内容 2. 选择微信号 3. 定时发布 | 持续触达 |
| 自动点赞 | 工作台 → 自动点赞 | 1. 配置点赞规则 2. 启用任务 | 增加互动 |
| 入群欢迎 | 工作台 → 入群欢迎语 | 1. 配置欢迎语 2. 选择群组 3. 启用 | 新成员欢迎 |
| 标签管理 | 流量池 → 流量标签 | 1. 创建标签 2. 给好友打标 | 精准分类 |

### 3.3 转化阶段

| 运营场景 | 系统功能 | 操作步骤 | 预期效果 |
|:---|:---|:---|:---|
| 消息群发 | 工作台 → 群消息推送 | 1. 选择目标 2. 编辑内容 3. 发送 | 批量触达 |
| 流量分发 | 工作台 → 流量分发 | 1. 选择流量池 2. 选择客服 3. 启用分发 | 自动分配 |
| 客户跟进 | 触客宝 → 聊天界面 | 1. 查看消息 2. 回复客户 3. 记录跟进 | 促成转化 |
| 数据分析 | 数据中心 → RFM分析 | 1. 查看客户分层 2. 制定策略 | 精准营销 |

## 六、风控规范

### 6.1 微信风控红线

| 风险行为 | 处罚 | 系统监控 |
|:---|:---|:---|
| 频繁加人 | 健康分-15，暂停24h | 健康分系统 |
| 消息轰炸 | 健康分-20，暂停24h | 群发监控 |
| 敏感词汇 | 警告/封号 | 内容审核 |
| 诱导分享 | 警告/封号 | 内容审核 |

### 6.2 健康分管理

| 健康分范围 | 状态 | 每日加人上限 | 建议操作 |
|:---|:---|:---|:---|
| 90-100 | 优秀 | 20人 | 正常运营 |
| 70-89 | 良好 | 15人 | 注意频率 |
| 50-69 | 一般 | 10人 | 减少操作 |
| 30-49 | 警告 | 5人 | 暂停加人 |
| <30 | 危险 | 0人 | 停止使用 |

### 6.3 内容审核标准

| 内容类型 | 允许 | 禁止 |
|:---|:---|:---|
| 产品介绍 | ✅ 真实描述 | ❌ 虚假宣传 |
| 案例分享 | ✅ 真实案例 | ❌ 伪造案例 |
| 促销活动 | ✅ 合规促销 | ❌ 诱导分享 |
| 行业内容 | ✅ 专业内容 | ❌ 敏感话题 |
